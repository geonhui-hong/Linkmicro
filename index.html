Pipeline Creation
You can designate the relationship between cells and the order of execution, as well as create a pipeline that includes key cells. Furthermore, you can edit and manage components consisting of pipelines.
Caching Management
When you execute a component, the results of its execution are cached. If the same cell is rerun, the existing results are reused to reduce work time. Furthermore, the cached results are stored and shared to maintain work consistency.
Powerful Collaborations
You can save and share not only the whole or part of the pipeline, but also the execution results. Furthermore, additional functions such as add comment, component grouping, python script conversion support efficient collaboration.
