Pipeline creation
In Link, you can specify the relationship between cells as well as the execution sequence. You can create a pipeline on the same screen as the code to improve readability.
Caching management
Link manages the result values of the cells that have already been executed separately from those that haven't. It also prevents work duplication and work-efficiency loss through caching.
Easily collaboration
You can save and share your outputs that are pipelines, components, caches, python codes. You can collaborate more comfortable with Link
